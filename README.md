# Bash Scripting Suite for System Maintenance

## Overview
A suite of Bash scripts to automate system maintenance tasks such as backups, updates, and log monitoring.

## Scripts
- `backup.sh` — Automated backups
- `update.sh` — System updates and cleanup
- `logmonitor.sh` — Monitor logs and alert / report conditions
- `menu.sh` — Menu-driven interface to run scripts

## Usage
```bash
chmod +x *.sh
./menu.sh
